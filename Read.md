# luthertech electronics  

ElectroHub is an online electronics company offering high-quality gadgets, accessories, and home electronics at affordable prices. Our goal is to make modern technology accessible, reliable, and easy to shop for.

##  Features
- Wide range of electronics (phones, laptops, accessories, and more)
- User-friendly and responsive interface
- Secure and convenient online shopping experience
- Fast and reliable customer support

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project showcases an online electronics platform designed for learning, demonstration, and future expansion into a fully functional e-commerce system.
